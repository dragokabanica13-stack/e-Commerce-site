Online Shop Assignment Jednostavan front-end web sajt za online kupovinu sa tri glavne stranice: proizvodi, login i checkout. Projekat je napravljen kao statičan sajt koristeći HTML, CSS i JavaScript.

Sadržaj Stranica proizvoda (prikaz artikala i dodavanje u korpu) Login forma sa osnovnom proverom korisničkog imena i lozinke Checkout stranica sa formama za dostavu i plaćanje Promo kod validacija Responsive dizajn (prilagođavanje manjim ekranima) Tehnologije HTML5 CSS3 JavaScript (vanilla) Struktura projekta login.html proizvodi.html checkout.html css/ style.css login.css proizvodi.css checkout.css js/ main.js images/ Pokretanje projekta Preuzmi ili kloniraj projekat. Otvori folder projekta u editoru. Pokreni fajl proizvodi.html u browseru (ili preko Live Server ekstenzije). Napomena: Početna stranica projekta je proizvodi.html.

Funkcionalnosti Proizvodi Prikaz više proizvoda sa cenama. Klik na Add to cart dodaje cenu proizvoda u trenutni iznos korpe. Klik na ikonicu korpe prikazuje trenutni iznos porudžbine kroz alert poruku. Login Login forma sa poljima za email i password. Osnovna validacija je implementirana u JavaScript-u. Test kredencijali: korisničko ime: admin lozinka: admin Checkout Forma za unos podataka za dostavu. Forma za unos podataka kartice. Dugme za primenu promo koda. Promo kodovi Podržani kodovi:

SAVE10 (10% popusta) SAVE15 (15% popusta) FREESHIP (besplatna dostava) Ako je kod nevažeći, prikazuje se odgovarajuća poruka.

Ograničenja trenutne verzije Nema backend integracije (nema realne autentifikacije ni obrade plaćanja). Podaci se ne čuvaju u bazi. Naručivanje i checkout su simulirani (frontend demonstracija). Autor Projekat je urađen kao studentski/vežba assignment za demonstraciju rada sa HTML, CSS i JavaScript osnovama.
